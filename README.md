1. Project 개요
   : Image에서 개인정보를 추출하여 개인정보로 판단되는 부분을 모자이크하여 Image 재생산

2. Project 의의
   : 개인정보 유출로 인한 피해를 막기 위한 기술의 개발 및 RestAPI를 통한 배포

3. Project 목표
   : Image로 부터 일어나는 개인정보 유출 피해의 최소화
